# How to use

## Installation

```
yarn add @xchainjs/xchain-polkadot
```

## Polkadot Client Testing

```
yarn install
yarn test
```

Following peer dependencies have to be installed into your project. These are not included in `@xchainjs/xchain-polkadot`.

```
yarn add @polkadot/api
```
